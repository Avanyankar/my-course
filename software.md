## Zapret

  [Zapret](https://github.com/bol-van/zapret) — свободное, открытое, автономное средство противодействия DPI, которое не требует подключения каких-либо сторонних серверов. Может помочь обойти блокировки или замедление сайтов HTTP(S), сигнатурный анализ TCP и UDP протоколов, например, с целью блокировки VPN.

  [Zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) — fork оригинального репозитория, настроенный под работу discord и youtube.

  [Github репозиторий](https://github.com/bol-van/zapret);
  [В начало](#программное-обеспечение).

## Window Subsystem for Linux

  Window Subsystem for Linux (WSL) — это инструмент, позволяющий одновременно получить доступ к возможностям Windows и Linux на компьютере с Windows. Из возможностей представляются: устанавка дистрибутивов Linux (например, Ubuntu, OpenSUSE, Kali, Debian, Arch Linux и т.д.), использование служебных программ и средств командной строки Bash из Linux непосредственно в Windows, без дополнительных трат ресурсов устройства.

  [Установка](https://apps.microsoft.com/detail/9P9TQF7MRM4R?hl=ru-ru&gl=US&ocid=pdpshare);
  [В начало](#программное-обеспечение).

  Рекомендуемый к установке дистрибутив для wsl: [Ubuntu](https://apps.microsoft.com/detail/9PDXGNCFSCZV?hl=neutral&gl=US&ocid=pdpshare).
