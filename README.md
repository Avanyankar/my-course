## Программа обучения

  Курс подразумевает чередование занятий, сначала теория, потом практика. При этом теория и практика иногда будут двигаться независимо друг от друга.

### Теоретический курс

1. Введение. Базовые концепции программирования
2. Архитектура компьютера
3. Системы счисления
4. Представление данных в компьютере
5. Основы компьютерной логики
6. Программное обеспечение
7. Первая программа на C++
8. Базовый синтаксис. Первые задачи

### Практический курс

1. Введение. Настройка системы. Приложения для курса

  Для того, чтобы выложить решённые задачи, студент должен в синхронизированный с оригинальным репозиторием свой форк выложить их в формате students/\{nickname\}/\{t/p\}-\{hw-cw\}\#\{номер презентации\}/\{\#номер задания\}..., например, первая задача из первого домашнего задания по теоретическому курсу будет в папке students в папке Vlad в папке t-hw-\#1 в папке \#1 файле main.txt.
